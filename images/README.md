# City Tower photos

The site currently uses temporary placeholder images served from a CDN. To show
the real City Tower photos, drop the files below into this folder using **these
exact filenames**. Each `<img>` slot in `index.html` is tagged with a matching
`data-photo="..."` attribute, so swapping the `src` to `images/<filename>` is a
one-line change per image.
