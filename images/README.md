# City Tower photos

Drop the photo files into this folder using **these exact filenames** (all
lowercase, hyphens instead of spaces, `.jpg` extension). Each `<img>` in
`index.html` already points at `images/<filename>`, so once the files are
committed here they render automatically — no further code change needed.

## Notes

- Do not upload `.DS_Store` (a hidden macOS file) — it isn't needed.
- If a file is a PNG, just rename the extension to `.jpg`; browsers render it fine.
- Recommended: landscape, ~1600 px wide, optimised (< 400 KB each).
